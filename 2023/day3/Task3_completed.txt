1. To view what's written in a file.
Ans :- cat <filename>
2. To change the access permissions of files.
Ans:- chmod 777 <filename>
3. To check which commands you have run till now.
Ans:- history
4. To remove a directory/ Folder.
Ans:- ***To remove directory*** -----> rm -rf <directory_name>/ rmdir <directory_name>
      ***To remove file*** ----------> rm -rf <filename>
5. To create a fruits.txt file and to view the content.
Ans:- vi fruits.txt 
      ***To view file content****
     -----> cat fruits.txt
6. Add content in devops.txt (One in each line) - Apple, Mango, Banana, Cherry, Kiwi, Orange, Guava.
Ans:- cat >> devops.txt
7. To Show only top three fruits from the file.
Ans:- head -3 <filename>
8. To Show only bottom three fruits from the file.
Ans:- tail -3 <filename>
9. To create another file Colors.txt and to view the content.
Ans:- vi file_colors.txt and cat file_colors.txt
10. Add content in Colors.txt (One in each line) - Red, Pink, White, Black, Blue, Orange, Purple, Grey.
Ans:- cat >> Colors.txt
11. To find the difference between fruits.txt and Colors.txt file.
Ans:- diff fruits.txt Colors.txt
