1) Create a new repository on GitHub and clone it to your local machine.

Ans:-    CREATING NEW DIRECTORY:-
         1) Navigate to top right corner click on profile  --> your repository --> on top right click on new 

         CLONING DIRECTORY:-
         1) yum install git -y
         2) git clone <repository_url>
         
2) Make some changes to a file in the repository and commit them to the repository using Git

Ans:-.   ADDING AND COMMITING FILE
         1) touch file
         2) git add file
         3) git commit -m "first commit"
         
3) Push the changes back to the repository on GitHub

Ans:-.   git push origin master
